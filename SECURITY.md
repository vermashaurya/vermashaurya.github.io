# Security Policy

## Supported Versions

This project is a continuously updated personal portfolio.  
Only the latest version deployed at:

👉 https://shaurya.online

is actively maintained and supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### How to Report

- Open a private security advisory on GitHub (preferred), or  
- Email: shauryaverma.tech@gmail.com

---

### What to Include

Please provide:
- A clear description of the issue  
- Steps to reproduce (if applicable)  
- Potential impact  

---

### What to Expect

- Acknowledgement within 48 hours  
- Investigation and fix depending on severity  
- Updates shared once the issue is resolved  

---

### Scope

This project is a static portfolio site (HTML, CSS, JavaScript).  
There is no backend or sensitive user data stored.

However, issues related to:
- XSS (Cross-Site Scripting)  
- Dependency vulnerabilities  
- Misconfigurations  

are still taken seriously.

---

### Disclosure

Please do not publicly disclose vulnerabilities until they have been reviewed and resolved.

---

Thanks for helping keep this project secure.
